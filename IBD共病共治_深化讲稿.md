## 一、IBD基础认知的现代视角

### 1.1 Crohn病（CD）与溃疡性结肠炎（UC）的本质区别

IBD不是单一疾病，而是一组以慢性、复发性肠道炎症为核心特征的异质性疾病谱。理解CD与UC的本质区别，是理解其共病差异的基础。

| 特征维度 | Crohn病（CD） | 溃疡性结肠炎（UC） |
|----------|--------------|-------------------|
| **病变范围** | 全消化道（口腔至肛门） | 局限于结肠和直肠 |
| **炎症深度** | 透壁性（全层受累） | 黏膜及黏膜下层 |
| **分布模式** | 跳跃性、节段性 | 连续性、从直肠向近端延伸 |
| **主要并发症** | 瘘管、脓肿、狭窄 | 中毒性巨结肠、结直肠癌 |
| **肉芽肿** | 非干酪样肉芽肿（30%~40%） | 无 |
| **遗传风险** | NOD2、ATG16L1突变为主 | HLA-DR、IL-10R突变 |
| **肠外表现（EIM）** | 更常见，尤其关节炎 | PSC关联性更强（UC > CD） |
| **手术后复发** | 高（吻合口附近） | 全结肠切除可根治 |

> **临床启示**：CD的透壁性炎症决定了更强的系统性炎症信号传播能力；UC与PSC的特殊关联提示独特的胆管免疫靶向机制。【NEW】

### 1.2 发病机制的三角模型（2024年更新视角）

```
        遗传易感性
         (多基因)
            ▲
           /|\
          / | \
         /  |  \
   环境因素 → 免疫失调 ← 肠道微生物失调
   (抗生素、饮食、  (Th1/Th17/Treg   (菌群多样性↓
    吸烟、卫生)      失衡)            致病菌↑)
```

**关键进展**【NEW】：
- **上皮屏障基因突变**（如 *LRRK2*、*CARD9*）直接影响肠道通透性
- **自噬缺陷**（ATG16L1 T300A 变异）损害抗原处理和细菌清除
- **Th17/Treg失衡**是EIM发生的关键免疫基础，IL-17A、IL-23轴参与多个靶器官损伤

## 二、共病的分类体系

> 本讲稿采用**机制驱动的四类分类法**，取代传统的器官系统分类，更清晰地展示共病与IBD的内在关联。

### 分类框架总览

```
IBD共病
├── I. 免疫介导型共病（直接由IBD炎症机制驱动）
│   ├── 关节病变（中轴型 + 外周型）
│   ├── 皮肤病变（结节性红斑、坏疽性脓皮病）
│   └── 眼部病变（葡萄膜炎、巩膜外层炎）
│
├── II. 器官特异性共病（免疫靶向特定器官）
│   ├── 肝胆系统：PSC vs 脂肪肝（机制不同）
│   ├── 代谢骨病：骨质疏松症（多因素打击）
│   └── 血管系统：血栓栓塞
│
├── III. 继发性共病（IBD病程/治疗的后果）
│   ├── 营养缺陷（B12、VitD、铁、叶酸）
│   ├── 感染风险（免疫抑制治疗相关）
│   └── 药物相关并发症（骨髓抑制、肝毒性）
│
└── IV. 心理社会共病（双向影响）
    ├── 焦虑障碍 & 抑郁症
    └── 健康相关生活质量损害
```

## 三、共病的中心病理生理学机制【新增机制章节】

> 这是本讲稿最核心的新增内容，阐明为什么IBD会导致如此广泛的系统性并发症。

### 3.1 肠道屏障功能障碍——"漏肠"级联反应

```
正常肠道上皮              IBD炎症状态
━━━━━━━━━━━          ↓↓↓↓↓↓↓↓↓↓↓
  完整紧密连接            紧密连接蛋白↓
  (claudin, occludin)    (claudin-1, ZO-1 ↓)
       |                       |
  阻止抗原入血              细菌产物入血
                          (LPS, 肽聚糖, 鞭毛蛋白)
                               |
                        激活全身免疫 → 关节/皮肤/肝脏
```

**关键分子机制**【NEW】：
- TNF-α和IFN-γ直接下调 claudin-1、ZO-1 表达，增加上皮通透性
- 细菌脂多糖（LPS）通过 TLR4-NF-κB 通路激活系统性炎症
- 微生物产物触发肠道外树突状细胞，启动靶器官免疫反应

*参考文献：Turner JR, Nat Rev Mol Cell Biol, 2009; Odenwald MA & Turner JR, Nat Rev Gastroenterol Hepatol, 2017*

### 3.2 微生物群落失调（Dysbiosis）——系统炎症的放大器

**IBD特征性菌群改变**：

| 微生物变化 | CD | UC | 系统后果 |
|------------|----|----|----------|
| 普拉梭菌（*F. prausnitzii*）↓ | ↓↓↓ | ↓↓ | 肠道保护减弱，Treg细胞减少 |
| 肠球菌属↑ | ↑↑ | ↑ | 激活NF-κB，促进关节炎 |
| 变形菌门↑ | ↑↑↑ | ↑↑ | 内毒素负荷增加 |
| 双歧杆菌↓ | ↓↓ | ↓ | 短链脂肪酸产生↓，屏障修复受损 |

【NEW】**Dysbiosis → EIM的机制链**：
1. 肠道菌群失调 → 异常代谢产物（次级胆汁酸、色氨酸代谢产物）
2. 激活 NLRP3 炎症小体 → IL-1β释放
3. 肠道 Th17 细胞激活 → 归巢至关节滑膜、皮肤、肝脏
4. 肠道-肝轴：细菌PAMPs通过门静脉激活肝脏Kupffer细胞 → PSC发病

*参考文献：Halfvarson J et al., Nat Microbiol, 2017; Nagalingam NA & Lynch SV, Inflamm Bowel Dis, 2012*

### 3.3 分子模拟现象——肠道抗原与靶器官的交叉反应

> 这是解释**免疫介导型EIM**最重要的机制之一。

**经典案例：IBD相关关节炎**

```
肠道细菌抗原（肠球菌、克雷伯菌）
         ↓ 分子模拟
HLA-B27 呈递的抗原表位
         ↓ 交叉反应
关节软骨 / 骶髂关节抗原
         ↓
异常CTL攻击
         ↓
关节炎/脊柱炎
```

**皮肤病变（坏疽性脓皮病）的分子模拟**【NEW】：
- 肠道上皮抗原与皮肤基底膜蛋白存在同源序列
- IBD患者皮损组织中检测到针对肠道上皮蛋白（BP-180同源物）的IgG自身抗体
- TNF-α作为共同效应因子驱动皮肤中性粒细胞功能失调

*参考文献：Ina K et al., J Gastroenterol, 1999; Conrad C et al., J Invest Dermatol, 2018*

### 3.4 持续T细胞激活与器官归巢——炎症信号的远端放大

**关键免疫细胞的器官定向迁移**【NEW】：

```
肠道淋巴组织
(Peyer斑、肠系膜淋巴结)
         ↓ T细胞初次激活
表达肠归巢受体 α4β7 + CCR9
         ↓ 但在炎症状态下：
同时上调 α4β1 + CCR6（关节归巢）
同时上调 α4β1 + CXCR3（皮肤归巢）
         ↓
多器官T细胞浸润
```

**治疗启示**：这解释了为什么靶向肠归巢（vedolizumab，阻断α4β7）对EIM疗效有限，而全身性TNF抑制剂（英夫利昔单抗）对关节炎/皮肤效果更好。

*参考文献：Feagan BG et al., NEJM, 2013; Lobatón T et al., J Crohns Colitis, 2014*

### 3.5 凝血系统异常——血栓风险的分子基础

**IBD相关血栓的多因素机制**：

```
炎症因子（TNF-α、IL-6、IL-1β）
         ↓
┌─────────────────────────────────┐
│ 1. 凝血激活：TF表达↑，FVII↑    │
│ 2. 抗凝系统抑制：蛋白C↓,S↓     │
│ 3. 血小板激活：活化聚集↑        │
│ 4. 纤溶抑制：PAI-1↑             │
│ 5. 内皮功能障碍：vWF↑           │
└─────────────────────────────────┘
         ↓
VTE风险较普通人群升高 2-3倍
```

【NEW】**Vedolizumab时代的新关注**：
- 抗整合素治疗可能改变淋巴细胞迁移模式，对凝血的影响仍在研究中
- 活动期IBD患者住院时VTE风险最高，需积极预防

*参考文献：Novacek G et al., Gut, 2010; Bollen L et al., J Crohns Colitis, 2019*

## 四、各类共病的核心特点与差异

### 4.1 关节病变——免疫介导型共病的代表

**【核心特点】** IBD相关关节病是最常见的EIM（发生率15%~30%），其独特之处在于关节炎与IBD肠道活动性的关系具有**类型特异性**。

#### 中轴关节炎（强直性脊柱炎/骶髂关节炎）

**核心机制**：
- **HLA-B27介导**（~50-70%患者HLA-B27阳性）
- 骶髂关节和脊柱的进行性炎症和骨化
- **与IBD活动性无关**——肠道缓解不等于关节症状缓解

**临床特征**：
- 慢性炎症性腰背痛（夜间痛、晨僵>30分钟）
- 对称性骶髂关节炎（影像学标准）
- 进展为脊柱强直

**与UC/CD的关联**：CD（5-10%）> UC（2-5%）

#### 外周关节炎

| 特征 | I型（少关节型） | II型（多关节型） |
|------|----------------|-----------------|
| 关节数 | <5个 | ≥5个 |
| 关节大小 | 大关节为主（膝、踝） | 小关节为主（指关节） |
| 与IBD活动性 | **密切相关** | **相对独立** |
| 破坏性 | 非破坏性，自限 | 可有侵蚀性 |
| HLA相关 | HLA-B27, DRB1 | HLA-B44 |
| 主要机制 | TNF-α驱动 | T细胞持续激活 |

**治疗核心差异**：
- I型：控制IBD即可改善关节症状
- II型：需要独立的关节靶向治疗（MTX、生物制剂）

> 【NEW】**2024 ECCO指南更新**：中轴关节炎推荐IL-17A抑制剂（塞库金单抗）需谨慎，因其可能加重IBD；TNF抑制剂（英夫利昔、阿达木单抗）仍是首选的IBD+中轴关节炎共治方案。

### 4.2 皮肤病变——分子模拟与中性粒细胞功能异常

**结节性红斑（EN）vs 坏疽性脓皮病（PG）**——两者机制截然不同：

| 特征 | 结节性红斑（EN） | 坏疽性脓皮病（PG） |
|------|-----------------|-------------------|
| 发生率 | 3%~10% | 0.6%~2% |
| 病理基础 | 皮下脂膜炎（中性粒细胞浸润） | 真皮中性粒细胞功能失调 |
| 主要机制 | Th1介导，TNF-α驱动 | 中性粒细胞过度激活，NF-κB↑ |
| 与IBD活动性 | **密切相关** | **可独立于IBD** |
| 好发部位 | 小腿伸侧（对称性） | 小腿/脚踝（溃疡性） |
| 疼痛特征 | 触痛性红色结节 | 剧烈疼痛，特征性紫罗兰色边界 |
| 治疗反应 | IBD控制→EN消退 | 需局部/全身激素±免疫抑制剂 |
| 手术禁忌 | 否 | **是（手术会加重PG——"同形反应"）** |

> 【NEW】**PG的特殊临床陷阱**：病理活检"阴性"不能排除PG，同形反应（pathergy）是其特征——清创手术可加剧溃疡扩大，临床鉴别至关重要。

### 4.3 肝胆系统——PSC与脂肪肝机制截然对立

#### 原发性硬化性胆管炎（PSC）

**【核心特点】** PSC是IBD最具特异性的肝胆共病，机制独特，进展风险高。

**发病机制**（肠-肝轴模型）【NEW】：
```
正常状态：      IBD炎症状态：
肠道屏障完整     屏障破坏 → 细菌PAMPs入门静脉
                          ↓
                 Kupffer细胞过度激活（TLR4/TLR9）
                          ↓
                 胆管周围炎症（"洋葱皮样"纤维化）
                          ↓
                 胆道上皮CCL25/CCR9归巢失调
                 (原本归巢肠道的淋巴细胞误入胆道)
                          ↓
                 PSC→继发性胆汁性肝硬化
```

**PSC的临床特殊性**：
- **70-80%的PSC患者合并UC**（几乎是UC独特的标志）
- UC合并PSC时，肠道炎症可以很轻微（甚至无症状）
- PSC患者的结直肠癌风险额外升高（需强化监控）
- **全结肠切除不能阻止PSC进展**（提示免疫触发点不完全在肠道）

#### MASLD/脂肪肝（代谢相关脂肪性肝病）

**与PSC机制对比**：

| 维度 | PSC | MASLD |
|------|-----|-------|
| 发生率（IBD） | 2%~7.5% | 15%~40% |
| 主要关联 | UC > CD | CD > UC |
| 核心机制 | 自身免疫/肠-肝轴 | 代谢失调/类固醇/营养 |
| 主要驱动因素 | 微生物归巢、免疫靶向 | 糖皮质激素、肥胖、胰岛素抵抗 |
| IBD治疗影响 | IBD活动↑ → PSC风险↑ | 激素使用→ MASLD风险↑ |
| 肝移植指征 | 是（终末期PSC） | 罕见（除非进展为MASH/肝硬化） |

### 4.4 骨质疏松症——多因素打击下的"沉默骨折"

**【核心特点】** IBD骨质疏松是多因素叠加的结果，不同因素的权重因患者而异。

**三重打击模型**：

```
                  ┌─── 慢性炎症 ───┐
                  │  TNF-α, IL-1β  │
                  │  IL-6, RANKL↑  │
                  │  破骨细胞活化  │
                  └────────────────┘
                          +
                  ┌─── 皮质激素 ───┐
                  │  成骨细胞↓     │
                  │  骨胶原合成↓   │
                  │  肠道钙吸收↓   │
                  └────────────────┘
                          +
                  ┌── 营养与吸收不良 ──┐
                  │  维生素D↓          │
                  │  钙吸收↓（CD回肠） │
                  │  体重↓             │
                  └────────────────────┘
                          ↓
             骨密度↓（腰椎、股骨颈）
             脆性骨折风险↑2-3倍（CD > UC）
```

**CD与UC的差异**【NEW】：
- CD骨质疏松风险**显著高于UC**，原因：
  1. 末段回肠受累→维生素D和钙吸收减少
  2. 更长的累积激素使用史
  3. 更广泛的营养不良
  4. CARD8/CARD9变异影响骨代谢调控

**监控建议（基于ECCO 2023）**：
- 诊断时即行基线DXA扫描
- 累积使用激素≥3个月者，每1-2年复查DXA
- 活动期IBD常规补充钙1000mg/d + VitD 800-1000IU/d

### 4.5 静脉血栓栓塞（VTE）——常被低估的风险

**【核心特点】** IBD患者VTE风险较一般人群升高2-3倍，活动期住院患者风险最高，但临床上预防不足。

**风险分层**：
- 活动期IBD（住院）：风险最高（OR 3.2-8.0）
- 缓解期IBD（门诊）：中等风险（OR 1.5-2.5）
- 手术围手术期：极高风险（需药物预防）

**机制核心**（Virchow三联征在IBD的体现）：
1. **高凝状态**：纤维蛋白原↑、FVIII↑、vWF↑、抗凝蛋白C/S↓
2. **血管壁损伤**：内皮细胞功能障碍（炎症因子直接损伤）
3. **血流缓慢**：住院制动、脱水、贫血

> 【NEW】**ECCO 2022指南推荐**：所有活动期IBD住院患者在排除出血风险后均应接受低分子肝素预防性抗凝；IBD合并活动期疾病时不应因担心消化道出血而常规拒绝VTE预防。

### 4.6 心理社会共病——双向调节轴

**【核心特点】** 心理共病不是IBD的"并发症"，而是与其共享神经-免疫-内分泌轴的双向疾病。

**双向肠-脑轴机制**【NEW】：

```
IBD炎症状态：
肠道炎症 → 细胞因子（TNF-α, IL-6）→ 血脑屏障通透性↑
                                    → 小胶质细胞激活
                                    → 5-HT合成↓
                                    → HPA轴异常激活
                                    → 焦虑/抑郁症状

心理应激：
焦虑/抑郁 → HPA轴激活（皮质醇↑）→ 肠道通透性↑
           → 自主神经功能紊乱    → 肠道菌群改变
                                 → IBD复发风险↑
```

**发生率和影响**：
- IBD患者抑郁症发生率：21%~25%（高于普通人群3倍）
- IBD患者焦虑障碍发生率：19%~35%
- 抑郁症是IBD缓解期复发最强的独立预测因子之一

**常用评估工具**：
- PHQ-9（抑郁筛查）
- GAD-7（焦虑筛查）
- IBDQ（IBD生活质量专用量表）
- Harvey-Bradshaw指数 + 心理评分联合评估

## 五、循证共治原则与多学科协作

### 5.1 以IBD为中心的共治模式

**核心原则**：IBD是驱动大多数共病的根本原因，有效控制IBD肠道炎症是共病管理的基石，但不能完全替代独立的共病治疗。

**治疗优先级矩阵**：

| 共病类型 | IBD控制的影响 | 独立治疗需求 | 优先推荐生物制剂 |
|----------|--------------|-------------|-----------------|
| 外周关节炎I型 | **直接改善** | 低 | 任何TNF抑制剂 |
| 中轴关节炎 | 部分改善 | **高** | 英夫利昔单抗 |
| 结节性红斑 | **直接改善** | 低 | TNF抑制剂 |
| 坏疽性脓皮病 | 部分改善 | **高** | 英夫利昔单抗 |
| 葡萄膜炎 | 部分改善 | **高**（眼科）| 阿达木单抗 |
| PSC | 关联弱 | **高**（肝科） | 无明确优势 |
| 骨质疏松 | 间接改善 | **高**（补充治疗） | 减少激素为目标 |
| 焦虑/抑郁 | 改善有限 | **高**（精神心理） | — |

### 5.2 多学科协作（MDT）框架

**核心团队**：
```
           消化科（IBD专科医生）
                  │
        ┌─────────┼─────────┐
        ↓         ↓         ↓
      风湿科    皮肤科    眼科
   (关节炎、  (EN、PG、  (葡萄膜炎、
    脊柱炎)   口腔溃疡)   巩膜外层炎)
        ↓         ↓         ↓
        └─────────┼─────────┘
                  │
        ┌─────────┼─────────┐
        ↓         ↓         ↓
      肝病科    骨科/内分泌   心理科
```

### 5.3 生物制剂的共病覆盖选择策略【NEW】

**基于共病谱的生物制剂首选推荐**：

| 患者特点 | 推荐生物制剂 | 原因 |
|----------|-------------|------|
| IBD + 中轴关节炎 | 英夫利昔单抗 / 阿达木单抗 | TNF抑制同时改善脊柱炎 |
| IBD + 皮肤病（EN/PG） | 英夫利昔单抗 | 最强的皮肤EIM证据 |
| IBD + 葡萄膜炎 | 阿达木单抗 | 眼科获批，穿透力强 |
| IBD + PSC | 维得利珠单抗或乌司奴单抗 | 肠道靶向，肝毒性低 |
| IBD + 结核风险 | 乌司奴单抗 / 维得利珠单抗 | TNF抑制剂结核风险高 |
| IBD + 焦虑/抑郁 | 任何有效控制IBD的制剂 | 炎症控制改善心理状态 |

> 【CAUTION】**IL-17A抑制剂（塞库金单抗）**：对银屑病性关节炎和强直性脊柱炎有效，但**可诱发或加重IBD**，有IBD病史者禁用或慎用。

### 5.4 监测随访的标准化体系

**年度随访推荐清单**（基于ECCO 2024指南）：

```
每次就诊：
├── IBD活动度评估（Mayo评分/HBI）
├── EIM症状问诊（关节、眼、皮肤）
├── 心理健康筛查（PHQ-9, GAD-7）
└── 用药安全性监测

每6个月：
├── 肝功能（ALP, GGT, ALT）
├── 全血细胞计数 + 铁代谢
└── 维生素D + B12水平

每1-2年：
├── DXA骨密度扫描（有危险因素者）
├── 结肠镜（合并PSC者强化间隔）
└── 空腹血糖 + 血脂（长期激素者）
```
