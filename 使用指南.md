# 聚类分析使用指南

## 概述

本项目提供了针对FC（粪便钙卫蛋白）和IBDQ（炎症性肠病问卷）数据进行聚类分析的完整R脚本。

## 快速开始

### 1. 安装R包

首先运行以下命令安装所需的R包：

```r
source("setup_packages.R")
```

这将自动安装以下包：
- tidyverse（数据处理和可视化）
- cluster（聚类算法）
- factoextra（聚类可视化）
- mclust（高斯混合模型）
- dbscan（DBSCAN算法）
- ggplot2（绘图）
- gridExtra（多图排列）
- tableone（基线特征表）

### 2. 准备数据

您的数据应包含以下变量：

**必需变量：**
- `FC_base`: 基线粪便钙卫蛋白值
- `FC_induc`: 诱导期粪便钙卫蛋白值
- `IBDQ_base`: 基线IBDQ评分
- `IBDQ_induc`: 诱导期IBDQ评分

**可选变量（用于基线比较和回归分析）：**
- `Age`: 年龄
- `Sex`: 性别
- `BMI`: 体重指数
- `Disease_Duration`: 病程
- `Outcome`: 结局变量（二分类）

参考 `sample_data.csv` 查看数据格式示例。

### 3. 修改脚本以加载您的数据

编辑 `clustering_analysis.R` 文件，将第32-48行的示例数据生成代码替换为您的实际数据：

```r
# 替换此部分为您的数据加载代码
data_raw <- read.csv("your_data_file.csv")
```

### 4. 运行分析

```r
source("clustering_analysis.R")
```

## 聚类方法

脚本实现了5种适合2变量聚类的常用方法：

### 1. K-means聚类
- **参数选择依据**: 肘部法则（Elbow Method）
- **优化指标**: 簇内平方和（WSS）
- **优点**: 简单快速，适合球形聚类
- **输出**: 肘部图、散点图

### 2. 层次聚类（Hierarchical Clustering）
- **参数选择依据**: 轮廓系数（Silhouette Analysis）
- **优化指标**: 平均轮廓宽度
- **优点**: 可视化树状图，不需要预先指定聚类数
- **输出**: 树状图、轮廓图、散点图

### 3. PAM聚类（Partitioning Around Medoids）
- **参数选择依据**: 轮廓系数
- **优化指标**: 平均轮廓宽度
- **优点**: 对异常值更稳健
- **输出**: 轮廓图、散点图

### 4. 高斯混合模型（GMM）
- **参数选择依据**: BIC准则（贝叶斯信息准则）
- **优化指标**: BIC值
- **优点**: 概率型聚类，可以处理椭圆形聚类
- **输出**: BIC图、散点图

### 5. DBSCAN
- **参数选择依据**: k-NN距离图
- **优化指标**: eps参数
- **优点**: 可识别任意形状的聚类和噪声点
- **输出**: k-NN距离图、散点图

## 分析流程

每种聚类方法独立执行以下步骤：

1. **参数优化**: 根据各聚类方法的评估指标选择最优参数
   - K-means: 通过肘部法则确定最佳k值
   - 层次聚类: 通过轮廓系数确定最佳k值
   - PAM: 通过轮廓系数确定最佳k值
   - GMM: 通过BIC准则确定最佳成分数
   - DBSCAN: 通过k-NN距离图确定最佳eps值

2. **执行聚类**: 使用最优参数进行聚类

3. **保存结果**: 将聚类结果保存到 `data_imputed` 数据框中

4. **可视化**: 生成散点图展示聚类结果

5. **模式分析**: 分析从基线到诱导期的聚类变化模式

6. **统计分析**: 
   - 生成基线特征比较表
   - 进行Logistic回归分析

## 两阶段聚类

分析在两个时间点执行聚类：

- **聚类1**: 基于标准化的 `FC_base_log` 和 `IBDQ_base`
- **聚类2**: 基于标准化的 `FC_induc_log` 和 `IBDQ_induc`

## 模式变化分析

脚本分析患者从基线到诱导期的聚类转换，创建模式组：

- **Stable_Low（稳定低风险）**: 保持在低风险聚类（如 1→1）
- **Improved（改善）**: 移动到较低风险聚类（如 2→1, 3→1）
- **Worsened（恶化）**: 移动到较高风险聚类（如 1→2, 1→3）
- **Stable_High（稳定高风险）**: 保持在高风险聚类（如 3→3）

## 自定义设置

### 调整最大聚类数

修改函数调用中的 `max_k` 参数：

```r
kmeans_cluster1 <- perform_kmeans_clustering(cluster1_data, "Cluster1_Base", max_k = 8)
```

### 调整DBSCAN参数

修改 `perform_dbscan_clustering` 函数中的参数：

```r
eps_value <- 0.3  # 根据k-NN图调整此值
minPts_value <- 5  # 形成聚类的最小点数
```

### 修改模式分组

编辑 `create_pattern_groups` 函数以定义基于临床解释的自定义模式分组。

## 结果解读

### 1. 参数选择图形

- **肘部图**: 寻找曲线的"肘部"点，即WSS下降速率开始变缓的位置
- **轮廓图**: 选择平均轮廓宽度最大的k值
- **BIC图**: 选择BIC值最高的成分数
- **k-NN距离图**: 寻找曲线的"拐点"确定eps值

### 2. 聚类散点图

- 不同颜色代表不同聚类
- 观察各聚类的分离程度和紧凑性

### 3. 转换分析

- 转换表显示从基线到诱导期各聚类之间的患者流动
- 识别常见的变化模式

### 4. 统计分析

- **基线特征表**: 比较不同模式组的基线特征差异
- **Logistic回归**: 评估模式组与结局的关联（单因素和多因素）

## 注意事项

1. 所有FC值在聚类前进行对数转换
2. 所有变量在聚类前进行标准化（z-score）
3. 脚本使用 `set.seed(123)` 确保结果可重复
4. 运行脚本前应处理缺失值（脚本中提供了占位符代码）
5. 模式分组可根据临床意义进行调整
